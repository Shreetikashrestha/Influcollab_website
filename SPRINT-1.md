Sprint 1 — Auth Design (Frontend)

Branch: auth-design (or sprint-1)

Guidelines:
- Create a branch named `auth-design` or `sprint-1`.
- Make small commits for each logical change (aim for ~10 commits):
  - Add Zod schema
  - Add login form component
  - Add register form component
  - Add dashboard page
  - Add home page
  - Install `zod`
  - Polish UI
  - Update README/SPRINT-1.md
  - Add video link
  - Final touchups

Deliverables (Sprint 1):
- Pages:
  - `/` (Home)
  - `/login` (Login Page)
  - `/register` (Register Page)
  - `/auth/dashboard` (dummy post-login page)
- Zod validation on forms
- Component separation under `app/(auth)/_components`
- Public GitHub repo link and unlisted YouTube video (3-5 minutes)

Video tips:
- Record 3-5 minutes showing the pages and validation errors
- Mention branch name and commit summary
- Keep language: English
