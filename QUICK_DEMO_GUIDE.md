### 4. CREATE CAMPAIGN (1.5 min)
- Login as brand

- Budget: $2000-$5000
- Add requirements and deliverables

### 5. APPLY TO CAMPAIGN (1 min)
- Login as influencer
- Browse campaigns

### 6. ACCEPT APPLICATION (1 min)
- Login as brand
- View applications
- Accept Sarah's application

### 11. ADMIN PANEL (1 min)

- Show statistics

### 12. RUN TESTS (30 sec)
- Terminal: `npm test`

### 13. FUTURE ENHANCEMENTS (30 sec)
- Show face

### 14. CONCLUSION (30 sec)
